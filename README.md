# Automated HL7 Security & Integrity Checks in Mirth Connect

This project automates validation and integrity checking of HL7 traffic inside Mirth Connect.

## Goals

✔ Verify HL7 messages have valid checksum or hash indicators  
✔ Detect tampering, corruption, or structural anomalies  
✔ Generate audit logs and optional alerts  

## Components

- JavaScript-based rule engine running inside Mirth Connect channels
- HL7 message parsing
- Validation rules
- Logging/audit mechanism

## Status

Repository initiated. Channel export and scripts are underway.
